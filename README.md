# Landing Page Auron Marketing

Landing page estática em HTML, CSS e JavaScript, inspirada no visual dark premium da referência enviada.

## Arquivos

- `index.html`: estrutura da página
- `styles.css`: visual completo, responsivo e animações
- `script.js`: menu mobile e animação de entrada

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Troque os links `https://wa.me/5500000000000` pelo WhatsApp correto.
3. Troque textos, números e segmentos conforme o cliente.
4. Suba os arquivos no GitHub, Vercel, Netlify ou hospedagem comum.

## Observação

A página foi criada com identidade própria para Auron Marketing, mantendo o estilo visual da referência, sem copiar logos, imagens de clientes ou dados da Brandcast.
